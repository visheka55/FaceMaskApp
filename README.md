# Face Mask Detection App 😷

This project detects whether a person is wearing a mask using AI.

## Features
- Real-time face detection
- Mask / No Mask classification
- Alert sound for no mask

## Technologies Used
- Python
- OpenCV
- TensorFlow / Keras
- Pygame

## How to Run
1. Install requirements:
   pip install -r requirements.txt

2. Run the app:
   python app.py

## Output
- Green box → Mask
- Red box → No Mask + Alert
